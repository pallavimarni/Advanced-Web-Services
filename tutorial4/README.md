## Run Locally

Clone the project

```bash
  git clone https://git.cs.dal.ca/marni/csci-5709-tutorials
```
Go to the Assignment1 directory

```bash
  cd tutorial4
```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
